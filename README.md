# Sistema de Gestão de Artefatos Scrum

Este projeto é um sistema de gerenciamento de projetos e tarefas Scrum. Baseado em **Django** (backend) e **React + Vite** (frontend).  

---

## Pré-requisitos

Antes de começar, instale as dependências globais:

- [Python 3.10+](https://www.python.org/)
- [Node.js 18+](https://nodejs.org/)

---

## Configurando o Backend (Django)

### 1️⃣ Entrar na pasta do backend
```bash
cd backend
python -m venv venv
source venv/bin/activate      
source venv/bin/activate       # Linux/Mac
venv\Scripts\activate          # Windows
pip install -r requirements.txt
python manage.py runserver 8000
```
